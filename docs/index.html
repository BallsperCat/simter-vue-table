Component simter-vue-table

Fork me on Github. Publish on npmjs

Define table tag by structured data.

Install :

// browser : (umd)
<script src="https://rawgit.com/simter/simter-vue-table/0.2.0/dist/simter-vue-table.min.js"></script>

// node :
import table from 'simter-vue-table'

Options :

| Name       | Description
|------------|--------------------
| columns    | Define table columns
|   id       | The column's id, it's the key in the `rows` item
|   label    | The column's visible text
|   width    | The column's width
|   children | Define how to group columns. The `id` and `width` will be ignored if define this property
| rows       | The table's row data

Example 1 : Simple Columns

// Html :
<st-table id="sample" :columns="columns" :rows="rows"></st-table>

// Js :
let table = window["simter-vue-table"].default  // browser
import table from 'simter-vue-table'            // node

new Vue({
  el: "#sample",
  data: {
    columns: [
      { id: "key1", label: "Sn", width: "50px" },
      { id: "key2", label: "X1", width: "100px" },
      { id: "key3", label: "X2", width: "50px" }
    ],
    rows: [
      { key1: 1, key2: "v1-1", key3: "v2-1" },
      { key1: 2, key2: "v1-2", key3: "v2-2" },
      { key1: 3, key2: "v1-3" }
    ]
  },
  components: {
    "st-table": table
  }
})
