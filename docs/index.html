Simter table vue component.

Fork me on Github

The target is to config the table by data config. Use together with simter-vue-colgroup and simter-vue-thead component.

// html
<st-table :columns="columns" :rows="rows"></st-table>

Example 1: Simple Columns

// vue data
{
  columns: [
    { id: "id1", label: "Column1", width: "100px" },
    { id: "id2", label: "Column2", width: "100px" },
    { id: "id3", label: "Column3", width: "100px" }
  ],
  rows: [
    { id1: "v11", id2: "v21", id3: "v31" },
    { id1: "v12", id2: "v22", id3: "v32" },
    { id1: "v13", id2: "v23", id3: "v33" }
  ]
}
If ignore `label` prop, use `id` prop instead.

// Result
| Column1 | Column2 | Column3 |
| v11     | v21     | v31     |
| v12     | v22     | v32     |
| v13     | v23     | v33     |
